About this prototype

This prototype explores some ideas for a next-gen stream:

1. Organize activity primarily by most-recently-annotated URL

2. Surface full thread context for recently-annotated URLs

3. Provide a leaderboard of most-recently-active users

4. Support compound tag filtering

5. Sketch some elements of a user profile page (first: activity stats)

Try it here: /stream.alt

This is not intended as a formal UX proposal but rather as a working demo of some of the mechanisms we need to put in place to make the stream a more social experience and a richer reflection of the annotation activity enabled by the product.
